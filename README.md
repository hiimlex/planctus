# Planctus

A single plant app with `React Redux` implementation

### Folders
```
|-src
|  |- app
|  |   |- routes
|  |   |- store
|  |   |   |- actions
|  |   |   |- reducers
|  |
|  |- ui
|  |  |- components
|  |  |- pages
|  |  |- styles
```

- **app**: my server app config
- **store**: redux store
- **routes**: app routes
- **ui**: user interface
- **components**: my specific components
- **pages**: all pages
- **styles**: global styles config
